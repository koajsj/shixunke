Car Rental Demo

汽车租赁系统

一个精简的 Spring Boot + JPA 示例页面，直接调用现有 REST 接口。

支付

ID 租赁ID 日期 金额 方式

运行说明

  • 启动项目后访问 http://localhost:8080/。
  • 页面所有数据都来自现有接口：/api/customers、/api/cars、/api/rentals、/api/payments。
  • 状态与支付方式表单使用英文枚举名，数据库内部仍会保存为中文值。
